# Wallpaper_Engine
一个便捷的创意工坊下载器
## 开源说明

本项目旨在练习golang 和 flutter 并减少劳动力。    
flutter build windows     




## 使用方法

**需要Chrome**    
下载地址：https://github.com/user1121114685/Wallpaper_Engine/raw/main/old/steamdownload.zip     

下载前请选择 wallpaper64.exe 的位置
